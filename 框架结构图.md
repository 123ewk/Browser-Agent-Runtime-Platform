# 三、状态机 + 事件示意

```
PENDING
   │
   ▼
PLANNING
   │
   ▼
RUNNING
   │
   ├─-> WAITING_USER (Semi-auto / 插话)
   │
   ├─-> RETRYING (异常 / selector失效)
   │
   └─-> RECOVERING (崩溃恢复)
   │
   ▼
COMPLETED / FAILED

```

**事件列表**：

- USER\_INTERRUPT → 暂停 & replanning
- CHECKPOINT\_RESTORE → 崩溃后恢复
- RETRY → selector / 页面操作失败
- RESUME → 用户中途允许继续执行
